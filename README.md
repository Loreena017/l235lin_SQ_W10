# Side Quest Week 10

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

### Controls

- **Move:** WASD
- **Shoot:** Arrow keys
- **Pause/Continue:** Spacebar
- **Skip to boss fight:** B (testing only)
- **Restart:** R (after winning or receiving a game over)

Explore the world, survive enemy waves while moving north, and enter the glowing boss zone to fight the giant orange blob. Use the minimap to track off-screen enemies and locate the apple power-up.

## Week 10 Changes

- Added an apple power-up that increases the player’s movement speed when collected.
- Added an indicator on the minimap showing where the power-up spawns.
- Added an on-screen timer showing how long the speed boost remains active.
- Changed the shooting control from the spacebar to the arrow keys, allowing players to aim and shoot more accurately in different directions.
- Added a pause function that allows players to pause and continue the game using the spacebar.
